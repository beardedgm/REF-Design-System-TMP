What tokens does the design system provide?
Colors, typography, spacing, border radii, shadows, z-index layers, and motion durations. Every visual property is tokenized and works across both dark and light themes.
Does this work without Tailwind?
Yes. The entire system is built on CSS custom properties defined in globals.css. Tailwind is an optional convenience layer. This showcase page uses zero Tailwind.
How do I add this to my project?
Import globals.css in your root layout, set data-theme on your html element, and use var() tokens in your CSS. With Tailwind, use arbitrary values like bg-[var(--bg-card)].
